# Smart-Contact-Manager-Project---Using-Object-Oriented-Programming
## Description: 
This Smart Contact Manager Project is a C++ based project that handles contact details provided by the user and performs all needed tasks as per the requirement. 
  <br>
  
## Features: 
- Add new contacts(multiple at a time, if needed)
- View contacts (by name, tag, first/last added)
- Update existing contacts(multiple at a time, if needed)
- Delete contacts(multiple at a time, if needed)
- Sort contacts by name or tags
- Export contacts to a text file
- View contact statistics
  <br>
  
## Technology used: 
- C++
- Object-Oriented Programming
- File Handling
  <br>
  
## Installation/Usage:
1. Clone the repository
2. Compile the Project using a C++ compiler
3. Run the executable
  <br>
  
## How to Run?
- Compile: g++ main.cpp -o SmartContactManager
- Run: ./SmartContactManager
